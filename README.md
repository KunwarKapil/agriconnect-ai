# AgriConnect AI – Smart Agriculture Management Platform

## Overview

AgriConnect AI is an AI-powered web platform designed for farmer groups and agricultural cooperatives. The platform helps manage farmer records, crop planning, weather monitoring, and agricultural decision-making through intelligent insights and AI-powered assistance.

## Features

- Farmer Management System
- Crop Planning & Harvest Management
- Weather Monitoring Dashboard
- AI Farm Advisor (Gemini API)
- Crop Disease Detection & Prevention
- Analytics & Reports Dashboard

## Tech Stack
